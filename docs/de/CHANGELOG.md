# Änderungsprotokoll

Alle bemerkenswerten Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/),
und dieses Projekt folgt [Semantic Versioning](https://semver.org/lang/de/).

## [1.1.0] - 2025-02-18

### Hinzugefügt
- Englische Übersetzung aller Dokumentations- und Skill-Dateien
- Mehrsprachige Repository-Struktur (Englisch im Root, Deutsch in `docs/de/`)
- Sprachauswahl-Hinweise im README

### Geändert
- Repository-Struktur für internationales Publikum reorganisiert

## [1.0.0] - 2025-02-15

### Hinzugefügt
- Erstveröffentlichung des Sokratische Methode Skills
- Vier Betriebsmodi: Pädagogisch, Therapeutisch, Juristisch/Analytisch, Coaching
- Sechs-Fragen-Taxonomie für umfassende Erkundung
- Implementierung der Sokratischen Zustandsmaschine
- Psychologische Sicherheitsmassnahmen
- Domänenspezifische Anpassungen für Mathematik, Therapie, Strategie und Textanalyse
- Umfassende Dokumentation mit Beispieldialogen
- Implementierungs-Checkliste für Qualitätskontrolle

### Features
- Automatische Modusauswahl basierend auf Nutzerkontext
- Frustrationsschwellen-Erkennung mit Gerüstunterstützung
- Bias-Balance-Mechanismen für dialektische Erkundung
- Ton-Management-Richtlinien zur Aufrechterhaltung der Beziehung
- Evidenzbasierte Fragetechniken aus CBT

### Dokumentation
- Vollständige SKILL.md mit theoretischer Grundlage
- README mit Installations- und Nutzungsanweisungen
- Mitwirkungs-Richtlinien
- MIT-Lizenz
- Beispieltranskripte für verschiedene Anwendungsfälle
