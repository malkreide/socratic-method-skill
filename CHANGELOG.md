# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-02-18

### Added
- English translation of all documentation and skill files
- Multilingual repository structure (English root, German in `docs/de/`)
- Language selection guidance in README

### Changed
- Repository structure reorganized for international audience

## [1.0.0] - 2025-02-15

### Added
- Initial release of the Socratic Method Skill
- Four operating modes: Pedagogical, Therapeutic, Legal/Analytical, Coaching
- Six-question taxonomy for comprehensive exploration
- Socratic State Machine implementation
- Psychological safety measures
- Domain-specific adaptations for Mathematics, Therapy, Strategy, and Text Analysis
- Comprehensive documentation with example dialogues
- Implementation checklist for quality control

### Features
- Automatic mode selection based on user context
- Frustration threshold detection with scaffolding support
- Bias balance mechanisms for dialectical exploration
- Tone management guidelines for maintaining rapport
- Evidence-based questioning techniques from CBT

### Documentation
- Complete SKILL.md with theoretical foundation
- README with installation and usage instructions
- Contributing guidelines
- MIT License
- Example transcripts for various use cases
